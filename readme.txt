PACKAGE DESCRIPTION

This package contains the MATLAB files containing the work presented in the thesis "Studies for Telescopic Span Morphing" by Tanvi Prakash submitted in partial fulfilment of the requirements for the degree of "Doctor of Philosophy".

%
%

CONTACT

Contact the author at www.tanviprakash.com before using any data.

%
%

The following files are part of the macro package:

  readme.txt    		This file
  TP Thesis R9.pdf	Theory reference
  Mastercode20.m		MATLAB 2023a version of main file (v20)
  Other .m files		Required for Mastercode to operate
  Ip.mat			Actuator Database required for FINDACT

%
%

GETTING STARTED

Unzip the folder to a working local directory
Run mastercode20.m from the MATLAB command window
All subroutines required are in the folder called by mastercode20.m
